Lumen | Undergraduate Thesis Project [2016-2017]

A two-part driver assistance application - a distraction detector to notify a driver when they aren't focusing on the road, and a navigation display to help them get where they need to go with the least distraction possible. The distraction detector is a pre-trained neural network that monitors the driver through a live video feed. The navigation is a web application that projects directions onto the windshield for the driver to follow.

Developed by a team of 4.

A Computer Vision project where me and colleague developed an application that tries to find hands/arms using our custom color-space for the skin tone, mask out the background, draw convex envelope around the object, and then tracks its path from the center of that object.

Distributed Image Viewer [2016]

A peer-to-peer distributed systems that allows users of different machines in the same network to send encrypted images to each other with a view counter and access privileges. Its core functionality is the following:
  1. Add/Delete/Download/Display embedded Images.
  2. Grant/Revoke Access to users in the database to view certain images with a set number of views allowed.
  3. View Downloaded/Allowed-Access Images' list.

Developed by a team of 4.
